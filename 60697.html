1. 什么是 Markdown?

Markdown 是一种轻量级的标记语言。它允许人们使用易读易写的纯文本格式编写文档，并支持图片、图表、数学公式等，然后转换成有效的 HTML 文档。 GitHub Flavored Markdown 语法（简称 GFM），下面我们来熟悉下常用标记说明。

2.标题

一级标题在行首使用一个#，二级标题使用两个##，以此类推。如下：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

3.样式

  • **粗体**用于表示粗体
  • *斜体*用于表示斜体
  • ~~删除线~~用于表示删除线

4.脚注

我们可以在这里插入一个脚注 [1]。

[^1]
[^1]: 这是脚注内容。

5.引用

> 这是一个引用

效果：

这是一个引用

8.2 无序列表

+、*、-都可以用来标识无序列表项。例如：

+ 项目1
* 项目2
- 项目3

输出结果：

  • 项目1
  • 项目2
  • 项目3

- [X] 已完成

输出结果：

  • [X] 已完成

9.代码

代码高亮：

9.1 Go

   func Fibonacci(n int) int {
       if n <= 1 {
           return n
       }
       return Fibonacci(n-1) + Fibonacci(n-2)
   }

输出：

func Fibonacci(n int) int {
    if n <= 1 {
        return n
    }
    return Fibonacci(n-1) + Fibonacci(n-2)
}

9.2 JavaScript

    function Fibonacci(num) {
        if (num <= 1) return 1;
        return Fibonacci(num - 1) + Fibonacci(num - 2);
    }

输出：

function Fibonacci(num) {
    if (num <= 1) return 1;
    return Fibonacci(num - 1) + Fibonacci(num - 2);
}

9.3 Lua

```lua
local function Fibonacci(n)
    local function doFibonacci(n, ret1, ret2)
        if (n <= 1) then
            return ret2
        end
        return doFibonacci(n - 1, ret2, ret1 + ret2)
    end
    return doFibonacci(n, 1, 1)
end
```

输出：

local function Fibonacci(n)
    local function doFibonacci(n, ret1, ret2)
        if (n <= 1) then
            return ret2
        end
        return doFibonacci(n - 1, ret2, ret1 + ret2)
    end
    return doFibonacci(n, 1, 1)
end

10.数学公式

LaTeX 语法的数学公式，例如：

$$
f(x) = a x^2 + b x + c
$$

会输出一个抛物线方程

$$
f(x) = a x^2 + b x + c
$$

而下面这个表达式

$$
F(\omega)=\int_{-\infty}^{+\infty} {f(t)e^{-i\omega t}dt}
$$

会输出一个傅里叶变换积分方程

$$
F(\omega)=\int_{-\infty}^{+\infty} {f(t)e^{-i\omega t}dt}
$$

要输出矩阵也很简单，只需要

$$
\begin{bmatrix}
1 & x & x^2 \\
1 & y & y^2 \\
1 & z & z^2 \\
\end{bmatrix}
$$

便可得到想要的效果

$$
\begin{bmatrix}
1 & x & x^2 \
1 & y & y^2 \
1 & z & z^2 \
\end{bmatrix}
$$

公式也可以显示在行内。例如 $f(x)=kx+b$ 就会输出 $f(x)=kx+b$，这是一个显示在行内的直线方程。

11.中文排版

11.1 开头空两格

&emsp;&emsp;我前面空了两格

效果：
  我前面空了两格

11.2 文字的居中

<center>这一行需要居中</center>

效果：

这一行需要居中

11.3 文字的字体和颜色

11.3.1 字体更换

<font face="楷体">我是楷体字</font>

效果：
我是楷体字

11.3.2 大小更换

<font face="楷体" size=10>我是楷体字</font>

效果：

我是楷体字

11.3.3 颜色替换

<font face="楷体" color=red size=10>我是红色楷体字</font>

效果：

我是红色楷体字
